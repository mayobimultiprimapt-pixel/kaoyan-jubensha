# 《101 失窃档案：红宝书之谜》
## 北京大学 101 政治阅卷组·剧本杀沙盒游戏完整设计方案

> **游戏定位**：剧本杀 × 沙盒探索 × 知识游戏化
> **人数**：4—6 人（1 主持人 DM + 3—5 玩家），也支持单人闯关模式
> **时长**：完整通关 6—8 小时；可分 5 日打卡（每日 1 幕 60—90 分钟）
> **覆盖范围**：桌面 docx 全部 44 章节 + 7 个形势政策专题，知识点全覆盖
> **学习目标**：把考研政治 101 的 21.9 万字颗粒化知识，嵌入沉浸式推理中，玩家"破案即背书"

## 一、世界观与主线

### 1. 背景设定
时值某年深秋，**北京大学燕园东北角，101 政治阅卷组数据归档处**。

归档处最深处的**「红宝书房」**里，收藏着一卷用微缩金箔镌刻的《红宝书》——它承载了考研政治**全部颗粒化知识点**，是阅卷组镇馆之宝。每年阅卷前，5 位模块主任须在馆长的注视下共同"请书"，合力开启三道密锁。

某日清晨，门卫发现大门上的三道密锁**全部熔为金水**，红宝书**不翼而飞**。但现场没有任何暴力撬动痕迹，无血、无影、无声——红宝书仿佛"自己走掉了"。

### 2. 玩家身份
你们是新入职的**"数据归档处超级研究员"**，由馆长亲自指派调查此案。你们被授予进入任意展区的通行权，但必须在 **5 天 5 夜**内复原红宝书，否则秋季阅卷季将被迫延期，整个 101 阅卷体系将陷入瘫痪。

### 3. 悬疑主线
- 5 位模块主任，人人都有作案嫌疑，人人都有不在场证明
- 但所有证明都自相矛盾——**矛盾本身，正是破案的关键**
- 随着调查深入，玩家会发现：红宝书从未被"偷"，它**自我解构**成了 **7+10+12+8+7 = 44 枚"知识徽章"**，散落在馆内各个展区
- **终章揭秘**：红宝书自愿散落，只因它的"守护者"——5 位主任之一——发出了一个秘密请求："**让知识走进人的心里，而不是躺在金箔上。**"玩家必须先"吃透"所有章节，才有资格重新锻造红宝书

### 4. 核心机制
> **知识即线索 · 背诵即解锁 · 推理即复盘**

玩家每进入一个"章节房间"，会遇见：
1. 一段**场景描述**（含该章核心历史/理论背景）
2. 一位**守护灵**（该章代表人物的幻影）
3. 一组**证物卡**（3—5 张，承载关键时间/著作/人物/论点）
4. 一道**机关谜题**（必须引用本章知识才能解开）

解谜成功 → 获得**知识徽章**；徽章集满 → 解锁下一幕；全徽章归位 → 重铸红宝书，真相大白。

### NPC 行为模式
- 每位主任**守护自己的模块展区**，玩家进入即触发对话
- 主任会主动抛出本章节的**钩子问题**（例：老马会问"你知道黑格尔辩证法的'合理内核'被谁颠倒过来的吗？"）
- 答对 → 主任友善、给线索；答错 → 主任冷淡、给假线索（沙盒机制）
- 5 位主任彼此有**交叉对话**，玩家互相串联才能拼出完整真相

## 五、终章 · 归位 · 馆长密室

### 1. 解锁条件
5 幕共 **44 枚徽章 + 3 条线索**全部集齐

### 2. 馆长登场
馆长是**老马、老毛、老史、老思、小时**五人中的一位（DM 根据玩家推理准确度随机决定真相版本），他/她道出真相：

> "红宝书从不属于金箔，而属于每一个愿意把它**读进心里**的人。三道密锁**不是被破坏**——它们是**被 5 位主任合力熔化**的，因为我们发现：把知识关在柜子里，就等于让它死去。于是我们决定：**让红宝书自我解构，把它藏进归档处的每一个角落**，等真正能读懂它的人把它重新拼回来。
>
> 现在，请你们把 44 枚徽章放进重铸炉……红宝书，归位。"

### 3. 终极机关
玩家需把 **44 枚徽章** 按**五模块·章序**归位到重铸炉的五个扇区。
归位正确→ 炉心燃起金光，红宝书**重新成形**，并且——**金箔上的每一个字都变成了玩家的笔迹**。
（隐喻：知识从"馆藏"变成了"我有"）

### 4. 结局彩蛋
**彩蛋一 · 阅卷权**：玩家获得"临时阅卷员"资格，馆长送来 3 份真实考研政治主观题答卷，玩家给分并写评语——DM 讲评。
**彩蛋二 · 红宝书的下一位守护者**：馆长低声："下一届考研人，需要一位新的归档处研究员……你们之中，谁愿意留下？"

## 七、证物卡系统（玩家装备）

每位玩家开局获得 **"研究员手账本"** 一本（实体道具，或电子版 PDF）。每通过一个房间抽取 1 张证物卡（共 44 张主卡 + 若干彩蛋卡），每张卡格式：

```
┌─────────────────────────────────┐
│ 证物卡 №013                      │
│ 《共产党宣言》初版扉页            │
├─────────────────────────────────┤
│ 出处：马原·第 1 章                │
│ 年份：1848 年 2 月 24 日·伦敦    │
│ 作者：马克思、恩格斯              │
│ 核心句："全世界无产者，联合起来！" │
│ 考点提示：★★★★★                 │
│ 关联徽章：「诞生徽章」            │
└─────────────────────────────────┘
```

**证物卡用途**：
1. **终章重铸**时须按模块扇区归位
2. **跨章联动**：某些证物卡（如"1848 宣言"+"1867 资本论"）组合后解锁隐藏剧情
3. **收集癖成就**：集满 44 张获得 DM 颁发的"红宝书研究员结业证"

## 八、游戏机制细则

### 1. 行动点（AP）系统
- 每位玩家每日开局获得 **8 AP**
- 进入房间耗 1 AP，答错谜题耗 2 AP，跨区移动 0 AP（沙盒）
- AP 用尽被"请出"归档处，次日重来（失败惩罚轻，避免阻塞）

### 2. 知识点题型
| 题型 | 占比 | 示例 |
|---|---|---|
| 填空 | 40% | "列宁在____年出版《____》，给物质下定义" |
| 单选 | 25% | "三大工人运动不包括：A 里昂 B 宪章 C 巴黎公社 D 西里西亚"（答 C） |
| 多选 | 20% | "马克思主义三大直接思想来源包括：（多选）" |
| 简答 | 10% | "用三句话概括《实践论》的核心" |
| 情境判断 | 5% | "若你是 1919 年北大学生，你的第一反应是：" |

### 3. 线索系统
- **明线索**（白卡）：房间正常奖励
- **暗线索**（红卡）：跨章节串联才能获得，关键真相依赖暗线索
- **伪线索**（黑卡）：答错时 NPC 给出，玩家需要辨伪

### 4. 合作与竞争
- **合作模式**：4—6 人共同通关，共享徽章
- **竞速模式**：分两队 PK，看谁先到终章
- **单人闯关**：沙盒自由探索，DM 由 AI 担任（可配合大模型 API）

### 5. 电子化扩展（可选）
- 如要做成电子沙盒游戏，推荐技术栈：
  - 前端：Unity 2D 或 RPG Maker MZ
  - 题库：从桌面 docx 抽取 1000+ 题（可脚本自动生成）
  - NPC 对话：接 Claude API，把本章原文作为 system prompt，NPC 即可"背诵"全部颗粒化内容

## 九、给 DM（主持人）的说明

1. **必读**：桌面 docx "《...超级研究员.docx》"全文 21.9 万字——每一章的"颗粒化考点"都是玩家可能提问的方向
2. **分章备忘**：DM 进入每个房间前，对应打开 docx 的章节段落，临场引用

4. **评分**：DM 根据玩家答题正确率给最终"阅卷等级"：
   - 90%+ → **特等研究员**（赠送印刷版红宝书周边）
   - 75—90% → **一等研究员**
   - 60—75% → **二等研究员**
   - <60% → **返聘见习**（下次再战）

### 守护灵登场
一位青年，胡须浓密，眼神炽热——**28 岁的马克思**浮现。

> 马克思："欢迎，年轻的研究员。要取走这房间的徽章，你们得先回答我——

### 3 步挑战
1. **第一步**（填空）："马克思主义产生的 **时代背景** 是 ____ 主义生产方式在西欧占主导，**阶级基础** 是欧洲三大工人运动——请说出三大工人运动的 **时间、地名、关键口号**。"
