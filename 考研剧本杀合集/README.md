# 考研剧本杀 · 联合归档处

四本桌面知识推理剧本杀（101/301/408/201）合集 + PWA 手机支持。

## PC 使用

直接双击桌面 `考研剧本杀·联合归档处.url` 即可（走 file:// 协议）。

## 手机使用

手机浏览器无法直接打开电脑 file://，需在 PC 上先启动一个本地 HTTP 服务。

### 1. 启动服务器

在 `C:\mq\` 下双击 **`启动手机服务.bat`**，窗口会显示本机局域网 IP，例如：

```
http://192.168.1.5:8080/考研剧本杀合集/
```

### 2. 手机连接

- 确保手机与电脑**同一 Wi-Fi**
- 手机浏览器粘贴上面的 URL

### 3. "添加到主屏" 变身原生 App

| 浏览器 | 步骤 |
|---|---|
| **iOS · Safari** | 底部分享按钮 → **添加到主屏幕** |
| **Android · Chrome** | 右上 ⋮ → **添加到主屏幕** / **安装应用** |
| **Android · Edge** | ⋮ → **添加到手机** |

添加后，主屏会出现**考研归档处**图标，点击后脱离浏览器全屏启动——和原生 App 一样。

### 4. 关闭服务

关掉 PC 上的 `.bat` 窗口即可。下次要用再双击一次。

> 升级后第一次启动：bat 会自动打开浏览器、终端里直接打印扫码二维码（指向第一张有效局域网 IP），并在 `C:\mq\` 下写出 `qr.png` 方便再扫。

## PWA 特性

- **离线可用**：每科首次访问后 Service Worker 预缓存 HTML/图标/manifest，断网也能继续玩。
- **跨科目独立模式**：5 份 manifest 共享根作用域（scope `../`），从启动器主屏跳任一科目不会退出独立窗口。
- **5 个独立 PWA 图标**：启动器 + 四科各有不同 `id` 和 icon，想单独把"101 政治"加主屏也可以。

> 旧版 PWA 升级提示：如果之前已经把旧版"考研归档处"加过主屏，建议 **删掉重新安装一次**，让浏览器读到新的 scope / PNG 图标。

## 存档

各科独立存档（浏览器 localStorage），同一台设备内互不影响。清除浏览器数据会丢失进度——想保留可使用游戏内"导出 PDF 通关纪念册"。

## BGM

每科场景/区域均有独立氛围 BGM（纯 Web Audio 合成，不占流量）。手机上点左下角 🎵 按钮启用。首次点击页面任意处激活音频。

## 常见问题

**手机打不开？** → 检查：① PC 上 .bat 还在运行？② 手机电脑同 Wi-Fi？③ 防火墙未拦截 8080？

**图标太小？** → iOS 的 SVG 图标可能显示为默认样式，属正常。

**进度丢了？** → 游戏内 🗑 按钮、或浏览器清理 localStorage。导出 PDF 可永久保存通关记录。
